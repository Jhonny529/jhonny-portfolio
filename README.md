# Jhonny Sanes — Backend Engineer Portfolio

Portfolio built with Astro, focused on backend engineering, software architecture, and production-ready systems.

## 🚀 Tech Stack

- Astro
- TypeScript
- Tailwind CSS

## 📦 Installation

```bash
pnpm install
````

## 🧪 Development

```bash
pnpm dev
```

## 🏗 Build

```bash
pnpm build
```

## 🔍 Preview build

```bash
pnpm preview
```
